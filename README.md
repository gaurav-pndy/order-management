# Order Management App

A full-stack order management feature built as part of a technical assessment.
The app allows users to browse a food menu, add items to a cart, place an order,
and track order status with simulated real-time updates.

---

## Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Router
- Context API

### Backend
- Node.js
- Express
- In-memory data store

### Testing
- Vitest + React Testing Library (Frontend)
- Jest + Supertest (Backend)

---

## Features

- View food menu with images
- Add/remove items from cart
- Checkout with customer details
- Place orders
- Track order status (Order Received → Preparing → Out for Delivery)
- Real-time updates via polling
- Clean, responsive UI


---

## Running the Project Locally

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
